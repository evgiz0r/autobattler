# Autobattler Game

A 1v1 autobattler game where you strategically place units to battle against an AI opponent.

## Play Now

**[Play the game here!](https://evgiz0r.github.io/autobattler/)**

## How to Play

- Choose units from 7 different tiers
- Place them in your build zone
- Watch them automatically battle the AI
- Earn gold to buy stronger units
- First to destroy the enemy core wins!

## Features

- 7 tiers of units with 4 types each (Melee, Ranged, Caster, Healer)
- Difficulty settings: Easy, Medium, Hard
- AI vs AI spectator mode
- Tier-based economy system
- Auto-tier unlocking every 5 rounds

Have fun!
